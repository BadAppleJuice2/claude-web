# 实用案例集

这里收集了一些使用 DeepSeek API 的实际案例，帮助你快速上手解决实际问题。

## 案例列表

| 案例 | 场景 | 难度 |
|------|------|------|
| [智能代码审查](code_review_bot.py) | 自动审查代码提交 | ⭐⭐ |
| [日志分析器](log_analyzer.py) | 分析服务器日志 | ⭐⭐ |
| [SQL生成器](sql_generator.py) | 自然语言转SQL | ⭐⭐⭐ |
| [API文档生成器](api_doc_generator.py) | 从代码生成文档 | ⭐⭐⭐ |
| [智能客服机器人](customer_service_bot.py) | 多轮对话客服 | ⭐⭐⭐ |
| [数据清洗助手](data_cleaner.py) | 清洗脏数据 | ⭐⭐ |

## 运行方式

```bash
cd examples/cases
pip install -r requirements.txt
export DEEPSEEK_API_KEY="sk-your-key"
python code_review_bot.py
```
