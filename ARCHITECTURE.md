## 组件说明

### 安装器层

| 文件 | 功能 | 技术 |
|------|------|------|
| `install.sh` | 环境检测、依赖安装、项目初始化 | Bash |
| `setup.sh` | 交互式配置 API Key | Bash（安装时生成） |
| `start.sh` | 服务启动、虚拟环境激活 | Bash（安装时生成） |

### Web UI 层

基于 [heng1234/claude-web](https://github.com/heng1234/claude-web)：

- **前端**: HTML + JavaScript + Tailwind CSS
- **后端**: Python FastAPI
- **数据库**: SQLite（会话历史）
- **通信**: WebSocket（流式输出）

### SDK 层

| 文件 | 功能 |
|------|------|
| `deepseek_sdk.py` | DeepSeek API 封装，简化调用 |
| `examples/` | 4 种典型场景的使用示例 |

### 容器化层

| 文件 | 功能 |
|------|------|
| `Dockerfile` | 镜像构建定义 |
| `docker-compose.yml` | 多容器编排 |
| `docker-entrypoint.sh` | 容器启动脚本 |

## 数据流

```
用户输入 → Web UI (FastAPI) → DeepSeekClient (SDK) → DeepSeek API → AI 响应
```

## 配置优先级

1. 环境变量 (`DEEPSEEK_API_KEY`)
2. `.env` 文件
3. `~/.claude/settings.json`
4. 默认值
